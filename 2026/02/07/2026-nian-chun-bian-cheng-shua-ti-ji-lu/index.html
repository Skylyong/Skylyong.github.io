3. Implement Long Short-Term Memory (LSTM) Network

题目：

Task: Implement Long Short-Term Memory (LSTM) Network
Your task is to implement an LSTM network that processes a sequence of inputs and produces the final hidden state and cell state after processing all inputs.

Write a class LSTM with the following methods:

init(self, input_size, hidden_size): Initializes the LSTM with random weights and zero biases.
forward(self, x, initial_hidden_state, initial_cell_state): Processes a sequence of inputs and returns the hidden states at each time step, as well as the final hidden state and cell state.
The LSTM should compute the forget gate, input gate, candidate cell state, and output gate at each time step to update the hidden state and cell state.

解题：

1） LSTM 的模型架构如下图所示：

2）对图中的递推公式的解释：

令 $x_t$ 为当前时刻输入，$h_{t-1}$ 为上一个时刻的隐藏状态，$c_{t-1}$ 为上一个时刻的单元状态：

  • 遗忘门（Forget Gate）:

$$
f_t = \sigma(W_f x_t + U_f h_{t-1} + b_f)
$$

  • 输入门（Input Gate）:

$$
i_t = \sigma(W_i x_t + U_i h_{t-1} + b_i)
$$

  • 候选单元状态（Candidate Cell State）:

$$
\tilde{c}t = \tanh(W_c x_t + U_c h{t-1} + b_c)
$$

  • 单元状态更新（Cell State Update）:

$$
c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t
$$

  • 输出门（Output Gate）:

$$
o_t = \sigma(W_o x_t + U_o h_{t-1} + b_o)
$$

  • 隐藏状态更新（Hidden State Update）:

$$
h_t = o_t \odot \tanh(c_t)
$$

其中

  • $\sigma$ 表示 sigmoid 函数（输出在0-1之间）
  • $\tanh$ 表示双曲正切函数 (输出在-1到1之间)
  • $\odot$ 哈达玛积（Hadamard product），即逐元素乘法

Simple Convolutional 2D Layer

题目：

In this problem, you need to implement a 2D convolutional layer in Python. This function will process an input matrix using a specified convolutional kernel, padding, and stride.

Example Input:

Output:

Reasoning:

The function performs a 2D convolution operation on the input matrix using the specified kernel, padding, and stride. The output matrix contains the results of the convolution operation.

解题：

下面是分步骤的详细解答说明：

1. 输入与参数定义

函数接收四个参数：

  • input_matrix ($X$): 输入的二维矩阵，尺寸为 $H_{in} \times W_{in}$。
  • kernel ($K$): 卷积核（滤波器），尺寸为 $K_H \times K_W$。
  • padding ($P$): 在输入矩阵周围填充 0 的圈数。
  • stride ($S$): 滑动窗口的步长。

2. 零填充 (Padding)

代码使用 np.pad 对输入矩阵的四周进行填充。

  • 操作：在高度和宽度的两侧各添加 $P$ 行/列的 0。

  • 数学变化：
    填充后的输入尺寸变为：

    $$
    H_{pad} = H_{in} + 2P
    $$

    $$
    W_{pad} = W_{in} + 2P
    $$

3. 计算输出尺寸 (Output Size) （推导附后）

在进行卷积之前，需要计算输出矩阵的形状。这取决于输入尺寸、卷积核大小、填充和步幅。

  • 代码对应：(padded_height - kernel_height) // stride + 1

  • 数学公式：
    输出的高度 $H_{out}$ 和宽度 $W_{out}$ 计算如下（其中 $\lfloor \cdot \rfloor$ 表示向下取整）：

    $$
    H_{out} = \left\lfloor \frac{H_{in} + 2P - K_H}{S} \right\rfloor + 1
    $$

    $$
    W_{out} = \left\lfloor \frac{W_{in} + 2P - K_W}{S} \right\rfloor + 1
    $$

4. 滑动窗口与卷积计算 (Convolution Loop)

代码通过双重循环遍历输出矩阵的每一个位置 $(i, j)$。

  • 定位窗口：
    根据步幅 $S$，在填充后的输入矩阵上找到对应的切片区域（Region）。区域的左上角坐标为：

    $$
    \text{start}_h = i \times S
    $$

    $$
    \text{start}_w = j \times S
    $$

    截取的区域大小与卷积核一致，即 $K_H \times K_W$。

  • 计算数值：
    代码执行 np.sum(region * kernel)。这包含两步：

    1. 逐元素相乘 (Element-wise Multiplication)：输入区域与卷积核对应位置相乘。
    2. 求和 (Summation)：将所有乘积相加得到输出矩阵在 $(i, j)$ 处的值。

  • 数学公式 (互相关)：
    输出矩阵 $Y$ 中的元素 $Y_{i,j}$ 计算公式为：

    $$
    Y_{i,j} = \sum_{m=0}^{K_H-1} \sum_{n=0}^{K_W-1} X_{pad}[i \cdot S + m, \ j \cdot S + n] \cdot K[m, n]
    $$

附： 卷积输出尺寸公式推导（Conv / Pooling）

1. 问题设定（以高度 $H$ 为例）

给定：

  • 输入高度：$H_{in}$
  • 卷积核高度：$K_H$
  • 填充（padding）：$P$
  • 步长（stride）：$S$

目标：求输出高度 $H_{out}$

宽度 $W$ 的推导完全对称，只需将 $H$ 替换为 $W$。

2. 第一步：Padding 后的有效输入长度

原始输入高度为：

$$
H_{in}
$$

上下各添加 $P$ 个 padding 后，总高度变为：

$$
H_{\text{padded}} = H_{in} + 2P
$$

3.第二步：卷积核“能放下”的条件

一个高度为 $K_H$ 的卷积核，要完整落在 padded 输入中，其起始位置必须满足：

$$
\text{起始位置} + K_H \le H_{in} + 2P
$$

即：

$$
\text{起始位置} \le H_{in} + 2P - K_H
$$

4. 第三步：卷积核的滑动方式

卷积核按步长 $S$ 滑动：

  • 第 1 次起始位置：$0$
  • 第 2 次起始位置：$S$
  • 第 3 次起始位置：$2S$
  • …
  • 第 $n$ 次起始位置：$(n-1)S$

5.第四步：最大可滑动次数

最后一次合法滑动满足：

$$
(n - 1)S \le H_{in} + 2P - K_H
$$

两边同时除以 $S$：

$$
n - 1 \le \frac{H_{in} + 2P - K_H}{S}
$$

$$
n \le \frac{H_{in} + 2P - K_H}{S} + 1
$$

由于 $n$ 必须为整数，取不超过该值的最大整数：

$$
H_{out} = \left\lfloor \frac{H_{in} + 2P - K_H}{S} \right\rfloor + 1
$$

6. 最终高度公式

$$
\boxed{ H_{out} = \left\lfloor \frac{H_{in} + 2P - K_H}{S} \right\rfloor + 1 }
$$

7. 宽度 $W$ 的推导（完全一致）

$$
\boxed{ W_{out} = \left\lfloor \frac{W_{in} + 2P - K_W}{S} \right\rfloor + 1 }
$$
