简介：Catppuccin 是一套以柔和色调为主的配色方案，分为四个口味（Latte / Frappé / Macchiato / Mocha），覆盖几乎所有主流工具。

总结

这套组合的核心理念是：每个工具只做一件事，但做到极致。

  • Ghostty 负责渲染终端，快且美观
  • fish 提供智能的命令行交互体验
  • Zellij 管理多窗口和会话
  • yazi 替代 cd + ls 的文件浏览场景
  • lazygit 把 git 工作流可视化
  • Catppuccin 把以上所有工具的视觉风格统一起来

整体下来，终端的使用效率和视觉体验都有了质的提升。
