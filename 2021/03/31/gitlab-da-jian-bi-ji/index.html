GitLab搭建笔记

发表于 2021-03-31 更新于 2021-04-01 分类于 爬坑总结

安装gitlab：

curl -s https://packages.gitlab.com/install/repositories/gitlab/gitlab-ee/
script.deb.sh | sudo bash

防火墙设置

之前已经配置好了，略

gitlab配置

打开/etc/gitlab/gitlab.rb

external_url 'http://gitlab.bytenlp.site'
# 系统端口冲突，我把端口改为了85
nginx['listen_port'] = 85

让配置生效，并重启gitlab服务

# 重新配置gitlab
sudo gitlab-ctl reconfigure
# 重启gitlab
gitlab-ctl restart

在浏览器输入”ip:端口号”查看效果，详细参考这里
